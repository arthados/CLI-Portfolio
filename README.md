# Karma Kansara Portfolio

Welcome to the source code of my personal portfolio. This project was built not just as a static landing page, but as an **interactive, CLI-driven experience** that showcases my background in building production-grade backend systems and scalable architecture.

## 🚀 The Vision
I wanted to build a portfolio that reflects my daily work environment. Instead of a traditional scrollable site, I implemented a **custom terminal interface** that allows users to interact with my project data, experience, and technical skills through command-line inputs.

## 🛠 Tech Stack
- **Frontend**: React.js, TailwindCSS
- **Interactive Terminal**: Custom hook-based CLI logic with state management
- **Build System**: Vite/Vercel

## 📦 Key Interactive Features
- **Interactive CLI**: Fully functional terminal emulator supporting commands like `whoami`, `experiences`, `projects`, `skills`, `contact`, and `clear`.
- **Dual-Mode**: Toggle between a terminal-centric view and a standard, professional web-view seamlessly.
- **Dynamic Content**: Data is decoupled from the UI, allowing for easy updates to experience and project descriptions.
- **Professional Impact**: Detailed technical breakdowns of my projects, emphasizing architecture and business-level problem-solving.

## 🚀 Deployment
This project is deployed on **Vercel**. 

### Local Development
1. Clone this repository: `https://github.com/arthados/CLI-Portfolio.git`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
